# Math Quiz Game - C++

A **console-based Math Quiz Game** developed in C++ for practicing basic arithmetic operations. This project helps improve programming skills while learning procedural programming concepts.

## Features

- Multiple levels of difficulty: Easy, Medium, Hard, or Mixed.
- Random arithmetic operations: Addition, Subtraction, Multiplication, Division, or Mixed.
- Immediate feedback for correct and incorrect answers.
- Tracks the number of right and wrong answers.
- Displays a pass/fail result at the end of the quiz.

## How to Play

1. Run the program in your C++ compiler (Visual Studio, Code::Blocks, etc.).
2. The program will generate a set number of random questions.
3. Enter your answer for each question.
4. The program will inform you if your answer is correct or not.
5. At the end, you will see the total right and wrong answers, and whether you passed the quiz.
